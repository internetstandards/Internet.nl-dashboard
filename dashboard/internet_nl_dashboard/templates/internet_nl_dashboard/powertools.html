About Users, DashboardUsers and Accounts

This system knows users, dashboardusers and account. This might be confusing at first, and this might help you getting started.

Anyone that can log into this system has a user account. The standard user account has been extended a bit: it makes it possible to become part of a larger 'Account'.

The larger Account holds the list with urls and has reports associated to it. This way multiple users can work on the same set of data.
